Get the stderr and stdout output of the IProcess.

Parameters
cancellationTokenThe CancellationToken for the operation.
Returns
A Task<TResult> resulting in the stderr and stdout output of the IProcess.

To guarantee that all data is received from the IProcess when redirecting streams to a file the result of this function must be awaited before IAsyncDisposable.DisposeAsync is called.
