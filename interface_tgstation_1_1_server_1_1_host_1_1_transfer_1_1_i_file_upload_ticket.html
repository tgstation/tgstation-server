Task<Stream> Tgstation.Server.Host.Transfer.IFileUploadTicket.GetResult ( CancellationToken  cancellationToken)

Gets the Stream for the uploaded file.

Parameters
cancellationTokenThe CancellationToken for the operation.
Returns
A Task<TResult> resulting in the uploaded Stream of the file on success, null if the ticket timed out.

The resulting Stream is short lived and should be buffered if it needs use outside the lifetime of the IFileUploadTicket.
