string? Tgstation.Server.Api.Models.Instance.AutoUpdateCron
getset

A cron expression indicating when auto-updates should trigger. Must be a valid 6 part cron schedule (SECONDS MINUTES HOURS DAY/MONTH MONTH DAY/WEEK). Empty string disables.

Updates will not be triggered if the previous update is still running. Incompatible with AutoUpdateInterval.
